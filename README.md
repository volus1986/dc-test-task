Run locally (dev):
1. Clone repository
2. docker compose build
3. docker compose up
4. open: http://localhost:3000 (Also you can see this link in console)

For Developing also run to install local dependencies:
npm install

Run prod
1. Clone repository
2. docker-compose -f docker-compose.prod.yml build
3. docker-compose -f docker-compose.prod.yml up -d
4. open site url
